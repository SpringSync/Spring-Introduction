## 운영 방식
- 1안 : 일주일 동안 정해둔 양을 공부하고 매주 한명씩 돌아가면서 발표하기
- 2안 : 일주일 동안 정해둔 양을 공부하고 매주 모임 당일에 제비뽑기로 발표자 선정
- 3안 : 일주일 동안 정해둔 양을 공부하고 파트를 분배하여 팀원 모두가 발표
  - ex) 1~3장을 팀원 모두가 공부하고, 팀원 A는 1장을, B는 2장을, C는 3장을 발표하는 형식

## Commit Convention
- 커밋 메시지는 섹션번호와 제목으로 구성한다.
- `#(섹션) 제목` 형태이며 `섹션` *뒤에 space가 있음*에 유의한다.
- 제목은 강의 주제를 기본으로 하되 상황에 맞게 수정 가능하다.
- 섹션에는 학습 중인 강의의 섹션을 작성하며 두자리 수로 작성한다.
- 예시
    - `#01 프로젝트 환경설정`
    - `#02 스프링 웹 개발기초`
    - `#03-1 회원도메인과 리포지토리 만들기`
